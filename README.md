# Credit Risk Default Prediction

## Project Overview
This project predicts whether a customer is likely to default on a loan using machine learning. It covers the complete data analytics workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model development, and business insights.

## Business Problem
Financial institutions need to identify high-risk borrowers before approving loans. This project analyzes customer financial data to understand the key factors influencing loan default and builds predictive models to support better lending decisions.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Machine Learning Models
- Logistic Regression
- Random Forest

## Key Findings
- Lower credit scores were associated with higher default risk.
- Higher interest rates increased the likelihood of default.
- Customers with lower incomes showed a greater tendency to default.
- Larger loan amounts were linked to slightly higher default rates.
